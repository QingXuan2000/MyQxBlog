QxBlog 部署指南

QxBlog 是一个面向技术写作的博客框架，默认主题为 QxPaper。

QxPaper 采用偏技术出版物的设计风格，强调排版、阅读节奏与长内容体验，而不是传统博客常见的卡片化 UI。

默认特性包括：

  • 深灰冷白配色
  • Typography-first 排版
  • GitHub 风格代码高亮
  • KaTeX 数学公式
  • Unified Markdown 渲染管线
  • 长文章阅读优化
  • GitHub Pages 自动部署

QxPaper 默认主题

QxPaper 是 QxBlog 内置的默认主题。

整体风格偏向：

  • Research Terminal
  • Technical Publishing
  • Markdown-native
  • Quiet Interface

适合：

  • 技术博客
  • 工程文档
  • 数学内容
  • AI / 系统设计
  • 长篇文章
  • 研究笔记

主题默认：

  • 使用 72ch 阅读宽度
  • 优化中英文混排
  • 支持亮暗色切换
  • 使用 GitHub 风格代码高亮
  • 使用 KaTeX 渲染数学公式

  • rehype
